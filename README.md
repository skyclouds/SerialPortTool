# SerialPortTool
串口通信工具支持Windows&amp;Mac
```shell
java -jar SerialPortTool.jar
==========串口工具==========
1.显示设备名称
2.退出
请选择:1
==========选择设备==========
1.cu.BLTH
2.tty.BLTH
3.cu.Bluetooth-Incoming-Port
4.tty.Bluetooth-Incoming-Port
5.cu.usbmodemLDYF000171781
6.tty.usbmodemLDYF000171781
0.返回
请选择设备:6
==========设置波特率==========
1.1200
2.2400
3.4800
4.9600
5.14400
6.28800
7.38400
8.19200
9.57600
10.115200
0.返回
请选择波特率:10
==========设置奇偶校验位==========
1.NONE
2.ODD
3.EVEN
4.MARK
5.SPACE
0.返回
请选择奇偶校验位:1
==========设置数据位==========
1.5位
2.6位
3.7位
4.8位
0.返回
请选择数据位:4
==========设置停止位==========
1.1位
2.2位
3.3位
0.返回
请选择停止位:1
==========发送数据==========
16进制字符串
0.返回
tty.usbmodemLDYF000171781-Send:010000
 [tty.usbmodemLDYF000171781-Send]010000
 [tty.usbmodemLDYF000171781-Read]read buffer=5(request=1024)
 [tty.usbmodemLDYF000171781-Read]0000020103
==========发送数据==========
16进制字符串
0.返回
tty.usbmodemLDYF000171781-Send:0
==========串口工具==========
1.显示设备名称
2.退出
请选择:2
谢谢使用再见!!!
```
